The Index servlet can receive the following query string parameters:
– clientmode: Chooses the client mode of the NemID JS client.
  Legal values: standard, limited.
  Default value: standard

– width: Width of the NemID JS client.
  Legal values: If parameter clientmode=standard, minimum 500px. If parameter clientmode=limited, minimum 200px (portrait) or 250px (landscape).
  Default value: If parameter clientmode=standard, 500px. If parameter clientmode=limited, 200px.

– height: Height of the NemID JS client.
  Legal values: If parameter clientmode=standard, minimum 450px. If parameter clientmode=limited, minimum 250px (portrait) or 200px (landscape).
  Default value: If parameter clientmode=standard, 450px. If parameter clientmode=limited, 250px.
